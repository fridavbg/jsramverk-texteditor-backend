# TextEditor

## Start appen

Första gången efter klonat github kör:

`npm i`

Starta appen

`npm start`

## Endpoints

### Main page
`/`

Displays main page

### Document page
`/docs`

GET request to show all documents within the database

POST request to insert one document

### 

### Database setup

`/docs/init`

Will insert 4 documents with lorem ipsum text
